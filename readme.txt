---make 방법과 실행파일 실행방법---
make main
./main

---메인메뉴---
1. Bring
data.txt 파일 이름을 입력하면 data.txt 파일에 있는 정보를 레코드로 받아들인다

2. Create
제품의 이름, 판매수, 원가, 인건비를 입력하면 레코드가 생성된다

3. Read
제품의 이름을 입력하면 그 제품의 정보를 출력해준다

4. List
현재 레코드에 저장된 모든 제품을 출력한다

5. Make Report
보고서 파일 이름과 보고자의 이름을 입력하면 현재 레코드에 저장된 제품 정보를 보고서로 생성한다

6. Total Profit
현재 레코드에 저장된 모든 제품들의 이윤을 합해서 출력한다

7. Update
제품 이름을 입력하면 그 제품의 정보가 업데이트된다

8. Delete
제품 이름을 입력하면 그 제품이 레코드에서 삭제된다

9. Save
현재 레코드에 저장된 제품의 정보를 new_data.txt 파일로 저장한다

10. Sort(By Profit)
현재 레코드가 저장된 제품의 정보가 높은 이윤 순서대로 정렬된다
